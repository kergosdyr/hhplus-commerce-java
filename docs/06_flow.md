## 잔액 충전

```mermaid
flowchart TB
    A(["Start"]) --> B["충전 요청"]
    B --> C{"유효한 사용자?"}
    C -- No --> D["오류 응답"]
    C -- Yes --> E{"요청 금액이 유효?"}
    E -- No --> F["오류 응답"]
    E -- Yes --> G["잔액 조회 및 충전"]
    G --> H(("잔액 충전 완료"))
    H --> I["정상 응답"]
    F --> J
    D --> J
    I --> J(["End"])

```

## 주문-결제

```mermaid
flowchart TB
    A(["Start"]) --> B[/"주문-결제 요청"/]
    B --> C{"유효한 사용자?"}
    C -- No --> D["오류 응답"]
    C -- Yes --> E{"잔액 충분?"}
    E -- No --> F["오류 응답"]
    E -- Yes --> G{"재고 충분?"}
    G -- No --> H["오류 응답"]
    G -- Yes --> I{"쿠폰을 사용하는가?"}
    I -- No --> J(("주문-결제 완료"))
    I -- Yes --> K{"유효한 쿠폰?"}
    K -- No --> L["오류 응답"]
    K -- Yes --> J
    D --> N
    F --> N
    H --> N
    L --> N
    J --> M["정상 응답"]
    M --> N(["End"])
```
